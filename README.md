


#TASKER247
Professional Email Signature Template
Installation & Deployment Guide

##1. Overview
This document provides step-by-step instructions for deploying the Tasker247 professional email signature template across Outlook (Desktop & Web) and Gmail.

The signature template has been designed to ensure:

Brand consistency

Professional visual identity

Mobile responsiveness

Standardized contact formatting

Embedded compliance-ready corporate messaging

##2. Package Contents
Your signature package should contain:

signature.html (Main signature file)

/images folder (Logo + icons, if applicable)

This README file

⚠ Important: Do not rename image files unless you update the image paths inside the HTML file accordingly.

##3. Pre-Deployment Checklist
Before installation, confirm:

Your name, title, phone number, and email address are correctly populated in the HTML file.

Logo file paths are correct.

Images are hosted online OR embedded (recommended for Gmail).

You are using Outlook Desktop (Windows/Mac), Outlook Web, or Gmail Web.

##SECTION A
Installing in Microsoft Outlook (Desktop – Windows)
Step 1: Save the Signature Files
Extract the signature package.

Place the folder in:

C:\Users\YourName\AppData\Roaming\Microsoft\Signatures
(If the “Signatures” folder does not exist, create it.)

Step 2: Verify Files
Ensure the following were created:

.htm file

.rtf file

.txt file

Images folder

If only an HTML file exists:

Open Outlook

Go to File → Options → Mail → Signatures

Create a new signature

Copy the HTML content into the signature editor

Step 3: Set as Default
Go to File → Options → Mail → Signatures

Select the Tasker247 signature

Set for:

New messages

Replies/Forwards

Click OK

#$SECTION B
Installing in Outlook Web (Office 365)
Log into Outlook Web.

Click Settings (Gear Icon).

Select Mail → Compose and Reply.

Open the HTML signature file in a browser.

Copy the full rendered signature (not the code).

Paste into the signature box.

Tick:

Automatically include on new messages

Automatically include on replies/forwards

Save.

##SECTION C
Installing in Gmail (Web)
Step 1: Prepare the Signature
Gmail does not accept raw HTML uploads directly. You must:

Open signature.html in a web browser.

Highlight the fully rendered signature.

Copy (Ctrl + C).

Step 2: Add to Gmail
Log into Gmail.

Click Settings (Gear Icon).

Click See All Settings.

Scroll to the Signature section.

Click Create New.

Name the signature:
Tasker247 Corporate

Paste the signature.

Assign to:

New Emails

Replies/Forwards

Scroll down and click Save Changes.

##SECTION D
Image Hosting Best Practice (Recommended)
For maximum compatibility across email clients:

Host images publicly (e.g., company website CDN).

Use absolute URLs (e.g., https://www.tasker247.com/images/logo.png).

Avoid linking to local drive images.

Outlook may block external images until trusted by recipient.

##SECTION E
Formatting Standards (Non-Negotiable)
To maintain corporate identity:

Do not modify font types.

Do not change logo size.

Do not add motivational quotes.

Do not insert additional banners.

Do not adjust brand colors.

Any customizations must be approved by Brand or Executive Office.

##SECTION F
Mobile Optimization Notes
Gmail Mobile App supports basic formatting.

Outlook Mobile may simplify layouts.

Avoid excessive image-based design.

Ensure font size is minimum 10–11pt equivalent.

SECTION G
Troubleshooting
Issue: Images not displaying
Confirm images are hosted online.

Check image URLs begin with https://

Ask recipient to “Download Images”

Issue: Signature formatting broken
Ensure you copied the rendered version, not HTML code.

Clear formatting in signature editor before pasting.

Avoid pasting via Word (causes styling corruption).

Issue: Signature too large
Compress logo to under 100KB.

Resize images to max width 300px.
